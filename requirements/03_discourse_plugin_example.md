# 功能需求：会员商品购买插件

## 概述
管理员在后台创建虚拟会员商品，普通用户可在前台浏览并模拟购买，  
购买成功后获得一定天数的会员身份。  
（本期不接入真实支付，购买为直接授权；支付集成为后期迭代）

## 角色

- **admin（管理员）**：创建、编辑、删除、激活/停用商品；查看购买记录
- **user（已登录用户）**：浏览商品列表，购买商品，查看自己的会员状态
- **anonymous（访客）**：可浏览商品列表，点击购买时跳转登录页

## 界面入口

- **admin**：后台 → 左侧菜单 → Plugins → "会员商品" 子菜单页（路径：`/admin/plugins/member-products`）
- **user**：顶部导航栏 → "会员中心" 链接（路径：`/member-center`）→ 商品列表
- **user 个人页**：`/u/用户名/membership` — 显示当前会员状态

## 操作流程

### Admin：创建商品

1. 进入 `/admin/plugins/member-products`，看到商品列表（初始为空）
2. 点击右上角 **"新建商品"** 按钮
3. 弹出表单，包含字段：
   - 商品名称（必填，文本）
   - 价格展示文字（如 "免费"，纯展示用，本期无真实支付）
   - 商品描述（多行文本，选填）
   - 授权天数（必填，数字，购买后会员有效天数）
   - 是否激活（开关，默认开启）
4. 点击 **"保存"**，商品出现在列表中

### Admin：编辑 / 停用商品

1. 在列表中点击某商品的 **"编辑"** 按钮，重新打开表单
2. 修改后保存
3. 切换"激活"状态：停用后，前台用户不再看到该商品
4. 点击 **"删除"** 需要二次确认对话框

### User：浏览和购买商品

1. 点击导航栏 **"会员中心"**，进入 `/member-center`
2. 看到所有已激活商品的卡片列表，每张卡片显示：名称、描述、有效天数
3. 点击某商品的 **"立即获取"** 按钮
4. 弹出确认对话框："确认获取 [商品名]？将为你的账号增加 [X] 天会员。"
5. 点击确认，系统为该用户增加会员天数
6. 对话框关闭，页面顶部显示成功提示

## 数据字段

### member_products 表（商品）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | integer | 主键 |
| name | string | 商品名称 |
| price_label | string | 价格展示文字（如"免费"）|
| description | text | 商品描述 |
| duration_days | integer | 授权天数 |
| active | boolean | 是否在前台展示，默认 true |
| created_at | datetime | 创建时间 |

### user_memberships 表（用户会员记录）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | integer | 主键 |
| user_id | integer | 关联用户 |
| product_id | integer | 关联商品 |
| purchased_at | datetime | 购买时间 |
| expires_at | datetime | 到期时间（purchased_at + duration_days）|

## 权限与业务规则

- 只有 admin 可以访问 `/admin/plugins/member-products`
- 访客点击"立即获取"跳转到 `/login`，登录后返回商品页
- 同一用户购买同一商品：**天数叠加**（expires_at 向后延伸），而非拒绝
- 商品被停用（active: false）后，前台不显示，但已购记录不受影响
- 商品被删除后，已购记录保留，仅 product_id 记录为已删除商品的 ID
- `duration_days` 必须为正整数，不能为 0 或负数

## 状态与反馈

| 场景 | 反馈 |
|------|------|
| Admin 创建商品成功 | 列表刷新，顶部绿色提示"商品已创建" |
| Admin 删除商品 | 弹出确认框，确认后列表移除该项，提示"商品已删除" |
| User 购买成功 | 顶部显示"获取成功！你的会员有效期至 YYYY-MM-DD" |
| User 购买失败（服务器错误）| 红色错误提示"操作失败，请稍后重试" |
| 访客点击购买 | 跳转到登录页，登录后自动返回 |
| 会员状态展示 | 用户个人页 `/u/用户名/membership` 显示当前状态："会员（有效期至 XX）"或"非会员" |
